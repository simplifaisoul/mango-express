# 🥭 Mango Express

A premium fresh mango e-commerce website built with modern web technologies.

![Mango Express](https://img.shields.io/badge/Mango-Express-f59e0b?style=for-the-badge&logo=react&logoColor=white)

## ✨ Features

- **12 Premium Products** — Fresh mangoes, dried fruits, juices, pickles, jams, and gift baskets
- **Shopping Cart** — Add/remove products, quantity controls, and checkout
- **Category Filtering** — Browse by Fresh, Processed, Drinks, or Gift Sets
- **Product Search** — Find products instantly
- **Responsive Design** — Optimized for desktop, tablet, and mobile
- **Smooth Animations** — Micro-interactions and hover effects throughout
- **Toast Notifications** — Feedback for cart actions
- **Newsletter Signup** — Stay updated with mango deals
- **Contact Form** — Get in touch directly

## 🛠 Tech Stack

- **React 19** — Latest React with hooks
- **TypeScript** — Full type safety
- **Vite 8** — Lightning-fast build tool
- **TailwindCSS v4** — Utility-first CSS with custom theme
- **Lucide React** — Beautiful icons
- **React Hot Toast** — Notification system

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
